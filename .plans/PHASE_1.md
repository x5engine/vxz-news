## Architecture
- **Framework:** React 18 (Vite)
- **Styling:** Vanilla CSS with Global Variables for "Dark Intelligence" theme.
- **Icons:** Lucide-React (Industry standard for clean, professional iconography).
- **Animations:** Framer Motion (For high-end transition feel).

## Design Philosophy
The VXZ.News interface is designed for **High-Density Information Consumption**. 
- **Dark Mode Only:** Reduces eye strain for long-duration analysis.
- **Grid-Based:** Inspired by tactical command centers and the Bloomberg Terminal.
- **Visual Hierarchy:** Critical status (Truth Score) is always globally visible.
